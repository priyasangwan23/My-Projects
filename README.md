# 🚀 Mini Projects Collection

A curated collection of small yet impactful frontend projects built to strengthen core web development skills and explore creative UI/UX concepts.

---

## 📌 Projects Included

### 🎮 Breakout Game
A classic arcade-style game where you control a paddle to break bricks using a bouncing ball.  
Focuses on game logic, collision detection, and canvas rendering.

---

### 🎨 Drawing App
An interactive drawing application that allows users to draw freely on a canvas.  
Includes features like color selection and brush size control.

---

### 🌊 Form Wave Animation
A visually appealing form with smooth wave-like label animations.  
Enhances user experience through creative UI transitions.

---

### 🖱️ Hoverboard
A fun interactive project where hovering over tiles creates glowing effects.  
Demonstrates event handling and dynamic styling.

---

### 🎵 Lyrics Search App
Search for song lyrics using an external API.  
Introduces API integration, async JavaScript, and dynamic content rendering.

---

### 🎬 Movie App
Browse and search movies with details fetched from an API.  
Focuses on working with APIs, UI design, and data handling.

---

### 🔄 Rotating Navigation Animation
A unique navigation menu with rotating animation effects.  
Explores CSS transforms and creative layouts.

---

### 💻 Terminal Style Landing Page
A developer-themed landing page designed like a terminal interface.  
Combines creativity with styling and layout skills.

---

### 🔀 Split Landing Page
A split-screen landing page with interactive hover effects.  
Highlights CSS transitions and responsive design.
